代謝症候群診斷標準

符合以下5項中的3項或以上即可診斷：

  • 腰圍異常
    男性 ≥ 90公分，女性 ≥ 80公分
  • 血壓異常
    收縮壓 ≥ 130mmHg 或舒張壓 ≥ 90mmHg
  • 空腹血糖異常
    空腹血糖 ≥ 100mg/dL 或糖化血色素 ≥ 5.7%
  • 三酸甘油脂異常
    三酸甘油脂 ≥ 150mg/dL
  • 好膽固醇異常
    男性 < 40mg/dL，女性 < 50mg/dL

糖尿病確診標準：

  • 空腹血糖 ≥ 126mg/dL
  • 糖化血色素 ≥ 6.5%

高危險群標準

符合以下1-2項代謝症候群標準：

  • 腰圍接近異常
    男性 85-89公分，女性 75-79公分
  • 血壓偏高
    收縮壓 120-129mmHg 或舒張壓 80-84mmHg
  • 空腹血糖偏高
    空腹血糖 100-125mg/dL 或糖化血色素 5.7-6.4%
  • 血脂異常
    三酸甘油脂 100-149mg/dL
  • 好膽固醇偏低
    男性 40-49mg/dL，女性 50-59mg/dL

建議：定期追蹤、生活型態調整、必要時藥物治療

×

正常健康標準

所有代謝指標均在正常範圍內：

  • 腰圍正常
    男性 < 90公分，女性 < 80公分
  • 血壓正常
    收縮壓 < 120mmHg 且舒張壓 < 80mmHg
  • 血糖正常
    空腹血糖 < 100mg/dL 且糖化血色素 < 5.7%
  • 血脂正常
    三酸甘油脂 < 150mg/dL
  • 好膽固醇正常
    男性 ≥ 40mg/dL，女性 ≥ 50mg/dL

建議：維持健康生活型態，定期健檢
